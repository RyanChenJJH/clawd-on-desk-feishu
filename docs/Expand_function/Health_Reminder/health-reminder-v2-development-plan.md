# 健康提醒第二版开发方案

> 前置：v1（[开发方案](health-reminder-development-plan.md) / [实施计划](health-reminder-stage-plan.md)）
> 已交付「不打断的共存式提醒 + 调度 + 静默 + clawd 一组动画 + 动画替换分区」。
> v2 在不破坏 v1 与上游兼容的前提下做**动画完善**与**体验增强**。
> 状态：规划。日期：2026-06-15

## 1. v2 目标

1. **动画完善（首要）**：补齐 cloudling / calico 专属健康动画全套，并扩充更多动画键。
2. **提醒预设库 + 导入导出**：一键添加常用提醒；提醒集可导入导出（复刻动画覆盖的 import/export）。
3. **音效**：每条提醒可选提醒音，接入既有音效替换体系。
4. **更聪明的调度（可选、默认关闭）**：仅在用户活跃（鼠标活动）时提醒、自适应间隔、
   DND 结束后顺延。
5. **气泡体验增强**：「全部知道了」、堆叠上限与排队策略可配、可拖动重定位、mini 模式适配。
6. **可选统计（本地、opt-in）**：完成/稍后次数、温和连续天数；绝不外发。
7. **国际化与无障碍**：提醒 UI 五语全覆盖；接入「减少动态」降级。

## 2. 动画完善（v2 重点）

### 2.1 三主题全套对齐

- 按[动画设计规范 §4](health-reminder-animation-design-spec.md) 的分镜，产出 cloudling、calico 的
  `drink/stretch/eat/offwork/eyerest` 专属动画，替换 v1 的回退素材。
- 三主题「同一动作的气质」对齐（云朵以形变、calico 以猫科语言、clawd 以像素动作）。

### 2.2 扩充动画键

新增键并写入各 `theme.json.healthReminders` 与设置下拉：

| key | 语义 | 备注 |
| --- | --- | --- |
| breathe | 深呼吸放松 | 极慢呼吸节律，配压力缓解类提醒 |
| posture | 端正坐姿 | 提醒挺直背部 |
| walk | 走动一下 | 角色横向小走（clawd 可复用 crabwalk 气质） |
| snack | 吃点水果 | 区别于正餐 eat |
| sleeptime | 该睡觉了 | 夜间就寝提醒，与 DND/静默协调 |

### 2.3 动画管线

- 复用 v1 的覆盖层与「健康提醒动画」替换分区，无需新机制。
- 新键即「加资源 + 加 theme.json 条目 + 加下拉项 + 加触发标签」，纯增量。

## 3. 提醒预设库与导入导出

- 预设库：在设置 Tab 提供「从模板添加」（喝水/久坐/护眼/午饭/下班/喝水进阶/番茄钟等），
  一键写入一条提醒。
- 导入导出：提醒集（不含任何敏感信息）导出为 JSON、可导入合并；复刻动画覆盖的
  `importAnimationOverrides/exportAnimationOverrides` 交互与 toast。

## 4. 音效

- 每条提醒 `sound` 接入既有 sound override / 主题 `sounds`；提供「预览」。
- 遵守现有 DND/静音策略（与 `previewSound` 的 skipped 语义一致）。

## 5. 更聪明的调度（默认关闭）

- `onlyWhenActive`：基于现有鼠标空闲判定（参考 state.js 的 mouseIdle/mouseSleep），离开电脑时不提醒。
- `adaptiveInterval`：连续「稍后」自动适度延长间隔，连续「知道了」恢复。
- `deferPastDnd`：DND/静默期间错过的提醒，在结束后顺延一次（带上限，避免堆积）。

## 6. 气泡体验增强

- 「全部知道了」一键清空所有健康气泡。
- 堆叠可视上限与排队策略可在设置调整（v1 固定 N=3）。
- 气泡可拖动重定位并记忆偏移。
- mini 模式下的健康气泡/动画适配（参考 theme.json `miniMode`）。

## 7. 可选统计（本地、opt-in）

- 仅本地记录每条提醒的 fired/confirmed/snoozed 次数与「今日已喝水 N 次」之类的温和计数。
- 默认关闭；提供「清空统计」；绝不联网、绝不进入任何远程通道。

## 8. 国际化与无障碍

- 提醒 Tab 与气泡文案五语（en/zh/zh-TW/ko/ja）全覆盖。
- 接入系统/应用「减少动态」：健康身体动画降级为低幅或静态首帧。

## 9. 兼容与迁移

- v2 全部为增量；v1 配置（`healthReminder`）向前兼容，新增字段带默认值。
- 新动画键缺素材时回退到 v1 行为（回退素材或仅文字气泡）。
- 默认关闭项（智能调度、统计）保持「未开启即零行为变化」。

## 10. 非目标（v2 仍不做）

- 云同步 / 跨设备同步提醒。
- 移动端独立健康提醒。
- 基于摄像头/可穿戴设备的生理数据。
- 任何把提醒内容外发到远程通道（飞书/Telegram）的行为——健康提醒是纯本地体验。

## 11. 验收（v2）

- cloudling/calico 五个动画专属化完成，三主题气质统一、无跳位。
- 至少新增 3 个动画键并可在提醒中选用、可在替换分区调整。
- 预设库可一键添加；提醒集可导入导出。
- 智能调度/统计默认关闭，开启后行为正确、关闭后零影响。
- 五语 UI 完整；「减少动态」降级生效。
- `npm test` 通过；v1 行为与上游兼容性不回归。
